# 实习小霸王 Intern King

一站式大学生实习求职平台 —— 实习岗位聚合 + AI 简历锐评，助你拿下心仪 Offer。

## 功能模块

### 实习广场
- 浏览实习岗位列表，支持按公司、城市、类型、分类筛选
- 收藏感兴趣的岗位，一键跳转投递

### AI 简历锐评
- 上传简历（PDF / DOCX），选择目标岗位
- AI 从匹配度、专业技能、项目经历、表达质量四个维度打分
- 结合网络面经搜索，给出针对性改进建议
- 毒舌风格点评，犀利但有用

### 个人中心
- 简历库管理（上传、下载、删除）
- 锐评历史记录
- 岗位收藏夹

## 技术栈

| 层 | 技术 |
|---|------|
| 框架 | Next.js 16 (App Router) |
| 语言 | TypeScript |
| UI | Tailwind CSS v4 + shadcn/ui |
| 认证 | Supabase Auth |
| 数据库 | PostgreSQL (Supabase) + Prisma 7 |
| 文件存储 | Supabase Storage |
| AI | DeepSeek API |
| 搜索 | Tavily Search API |
